External Code Sources

The following external code sources were used in this website, modified and adapted for this project:

Font Awesome Icons

Source: Font Awesome v6.0.0-beta3

Usage: All icons throughout the website for visual enhancement and navigation cues.

Attribution: Font Awesome by Dave Gandy - https://fontawesome.com/license (Free for commercial use)

Modifications made: Icons are used as-is with custom styling (color, size, hover effects) in CSS.

Base CSS Reset & Structure

Source: Adapted from tutorial content provided in INST0087 module materials and MDN Web Docs

Usage: Basic CSS reset, box-sizing, and layout principles.

Attribution: Based on concepts from MDN Web Docs (https://developer.mozilla.org) and module tutorial sheets.

Modifications made: Completely customized color scheme, typography, animations, and responsive breakpoints.

Accessibility Improvements

Source: WAVE Web Accessibility Evaluation Tool feedback and WCAG 2.1 guidelines

Usage: ARIA labels, skip links, keyboard navigation, and color contrast adjustments.

Attribution: Recommendations from WebAIM (https://webaim.org) and W3C WAI (https://w3.org/WAI/)

Responsive Design Patterns

Source: CSS-Tricks guides on Flexbox and Grid (https://css-tricks.com)

Usage: Card grid layouts, navigation bar responsiveness.

Modifications made: Custom breakpoints at 700px, adapted grid columns for content-specific needs.
